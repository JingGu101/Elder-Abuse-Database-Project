Database Usage

>> "Main" to return to homepage of Forensic Center Referral Form into the database.


>> "New Record" to manually add a record to the database


>> "Import Form" to import a Forensic Center Referral Form into the database.


>> "Import Excel" to insert excel file to database


>> "Search by" to query for clients in the database. The default will to show all. Refine the search by entering addtional search criteria.


>> "Cases status" to show the status of the case.
